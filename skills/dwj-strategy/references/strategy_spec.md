# 趋势游侠策略 — 完整规格

## 核心思路
板块轮动 + 趋势跟踪：做低位板块的轮动上涨趋势收益，本质上自己配一个板块ETF。

## 指标定义

### BBI（多空指标，短线资金线）
BBI = (MA3 + MA6 + MA12 + MA24) / 4

### MA60（长线资金线）
标准60日均线

### KDJ
标准KDJ(9,3,3)，重点关注J值

### 单针下30指标
```
N1 = 3   {短期周期}
N2 = 21  {长期周期}
短期 = 100 * (C - LLV(L, N1)) / (HHV(C, N1) - LLV(L, N1))
长期 = 100 * (C - LLV(L, N2)) / (HHV(C, N2) - LLV(L, N2))
```

## 零、前置排除：长阴放量

在所有选股逻辑之前，先排除近期有大资金撤退信号的标的。

### 排除条件（近5个交易日内出现任一天满足）
1. 阴线（收盘 < 开盘）
2. 跌幅 ≥ 1.5%
3. 成交量 ≥ 前20日均量的2倍

### 逻辑
长阴放量意味着大资金撤走很多，不确定性很大，短期不宜介入。
例如：读者传媒(2/12跌4.77%放量2.17x)、播恩集团(2/24跌1.95%放量3.18x)。

## 一、主策略：中长线趋势

### 趋势判断
- 上升区间：N型波段上升，低点抬高 + 高点抬高
- BBI > MA60（短线资金线在长线之上）
- 最好刚产生BBI上穿MA60的金叉

### 买入条件（全部满足）
1. 处于上升区间
2. J < 13（越低越好）
3. 收盘价 > BBI
4. BBI > MA60

### 买入执行
- 只在开盘/尾盘10分钟操作
- 买入价压日内分时均线（VWAP近似）
- 无加仓，买入即该标的全仓，后续只有减仓

### 止损
- 买入价往下5个价位，破位全清
- 利润全亏完 → 全清（只输利润）

### 止盈
- 第一档：累计盈利≥7%，卖出1/2
- 持有规则：不破BBI一直持有，破BBI清仓
- 盘中止盈：盘中累计+7%可触发，卖1/3或1/2

## 二、补充策略：单针下30

### 选股条件（全部满足）
1. 今日长期 ≥ 85（趋势没坏）
2. 今日短期 ≤ 30（急跌下探）
3. 昨日长期 ≥ 85（昨天也在高位）
4. 昨日短期 ≥ 85（昨天强势）

### 上升趋势定义（MA多头排列）
与主策略的N型判断不同，单针下30通道使用更严格的均线排列：
1. **完美多头排列**：MA5 > MA10 > MA15 > MA30 > MA60
2. **五条均线斜率均为正**：每条均线的今日值 > 昨日值

两个条件同时满足才算"上升趋势"。

### 使用条件
- 前提：MA多头排列上升趋势
- 做一根就撤（超短线）
- 止损：买入前一天收盘价
- 多标的：量比降序，量比≥8优先

## 三、补充策略：BBI金叉 + 量价齐升 + 缩量回调

### 选股逻辑（三步检测）
1. **BBI金叉**：最近20个交易日内，出现BBI上穿MA60
2. **金叉后量价齐升**：金叉点到波段高点之间，成交量放大（≥金叉前均量的1.5倍，越大越好），同时价随量升
3. **近期缩量回调**：高点之后价格回调，但回调时成交量相较上涨段大幅萎缩（≤60%，最好半量以下），且越回调量越小（地量地价）

### 核心判断
- 这是"主力建仓→洗盘→即将启动"的经典形态
- 放量倍数越大 → 主力建仓越积极 → 加分
- 缩量比例越低 → 洗盘越充分、抛压越小 → 加分
- 回调期间量逐步递减（后半段均量 ≤ 前半段均量）→ 地量确认

### 与其他通道的关系
- 独立的第三条选股通道，与主策略、单针下30并列
- 如果同一只票已经被主策略或单针下30选中，不重复进入

## 四、选股排序

### 板块聚类
- 对候选票做板块归类
- 同板块多个信号 = 板块启动，权重提升

### 排序规则
1. 量比降序
2. 量价齐升（价格上涨 + 量比 > 1）
3. 量比 ≥ 8 优先标记
4. 双信号（主策略+单针下30）优先

## 五、框架公式
总收益 = ∫(阳率 × 盈亏比)
- 阳率：买入后上涨概率（多维度叠加提升）
- 盈亏比：止盈止损规则决定（优先保盈亏比）
