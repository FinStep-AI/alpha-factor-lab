# 邓文晋趋势策略 — 完整规格

## 核心思路
板块轮动 + 趋势跟踪：做低位板块的轮动上涨趋势收益，本质上自己配一个板块ETF。

## 指标定义

### BBI（多空指标，短线资金线）
BBI = (MA3 + MA6 + MA12 + MA24) / 4

### MA60（长线资金线）
标准60日均线

### KDJ
标准KDJ(9,3,3)，重点关注J值

### 单针下30指标
```
N1 = 3   {短期周期}
N2 = 21  {长期周期}
短期 = 100 * (C - LLV(L, N1)) / (HHV(C, N1) - LLV(L, N1))
长期 = 100 * (C - LLV(L, N2)) / (HHV(C, N2) - LLV(L, N2))
```

## 一、主策略：中长线趋势

### 趋势判断
- 上升区间：N型波段上升，低点抬高 + 高点抬高
- BBI > MA60（短线资金线在长线之上）
- 最好刚产生BBI上穿MA60的金叉

### 买入条件（全部满足）
1. 处于上升区间
2. J < 13（越低越好）
3. 收盘价 > BBI
4. BBI > MA60

### 买入执行
- 只在开盘/尾盘10分钟操作
- 买入价压日内分时均线（VWAP近似）
- 无加仓，买入即该标的全仓，后续只有减仓

### 止损
- 买入价往下5个价位，破位全清
- 利润全亏完 → 全清（只输利润）

### 止盈
- 第一档：累计盈利≥7%，卖出1/2
- 持有规则：不破BBI一直持有，破BBI清仓
- 盘中止盈：盘中累计+7%可触发，卖1/3或1/2

## 二、补充策略：单针下30

### 选股条件（全部满足）
1. 今日长期 ≥ 85（趋势没坏）
2. 今日短期 ≤ 30（急跌下探）
3. 昨日长期 ≥ 85（昨天也在高位）
4. 昨日短期 ≥ 85（昨天强势）

### 使用条件
- 前提：上升波段区间
- 做一根就撤（超短线）
- 止损：买入前一天收盘价
- 多标的：量比降序，量比≥8优先

## 三、选股排序

### 板块聚类
- 对候选票做板块归类
- 同板块多个信号 = 板块启动，权重提升

### 排序规则
1. 量比降序
2. 量价齐升（价格上涨 + 量比 > 1）
3. 量比 ≥ 8 优先标记
4. 双信号（主策略+单针下30）优先

## 四、框架公式
总收益 = ∫(阳率 × 盈亏比)
- 阳率：买入后上涨概率（多维度叠加提升）
- 盈亏比：止盈止损规则决定（优先保盈亏比）
