# 技术分析详细参考

## 分析框架

### 一、趋势分析

#### 均线系统
| 均线 | 周期 | 用途 |
|------|------|------|
| MA5 | 周线 | 短线攻击线 |
| MA10 | 半月线 | 短线生命线 |
| MA20 | 月线 | 中线多空线 |
| MA60 | 季线 | 中线趋势线 |
| MA120 | 半年线 | 牛熊分界线 |

**均线形态：**
- **多头排列**: MA5 > MA10 > MA20 > MA60（趋势向上）
- **空头排列**: MA5 < MA10 < MA20 < MA60（趋势向下）
- **均线缠绕**: 均线交织粘合（方向选择期）

---

### 二、买卖点信号

#### 指标交叉
| 信号 | 条件 | 强度 |
|------|------|------|
| MACD金叉 | DIF上穿DEA，零轴附近或下方 | 买入 |
| MACD死叉 | DIF下穿DEA，零轴上方 | 卖出 |
| MACD二次金叉 | 零轴上方二次金叉 | 强买 |
| KDJ低位金叉 | K上穿D，三线均在20以下 | 强买 |
| KDJ高位死叉 | K下穿D，三线均在80以上 | 强卖 |

#### 突破买点确认
1. 放量（量比>1.5）
2. 收盘价站稳突破位
3. 次日不跌破突破位

#### 回踩买点确认
1. 缩量回调
2. 下影线支撑
3. 不破关键支撑位

#### 背离信号
- **底背离（买入）**: 股价创新低，MACD/RSI/KDJ不创新低
- **顶背离（卖出）**: 股价创新高，MACD/RSI/KDJ不创新高
- 一次背离关注，二次背离警惕，三次背离强烈信号

---

### 三、量能分析

#### 量价配合
| 价格 | 成交量 | 含义 |
|------|--------|------|
| 涨 | 放量 | 上涨有力 |
| 涨 | 缩量 | 上涨乏力 |
| 跌 | 放量 | 主力出货 |
| 跌 | 缩量 | 洗盘信号 |

#### 量能异动
- 量比 > 2.0: 异常放量
- 量比 < 0.5: 异常缩量
- 换手率 > 10%: 高度活跃
- 换手率 < 1%: 交投清淡

#### 资金流向
| 价格 | 资金 | 判断 |
|------|------|------|
| 涨 | 流入 | 真突破 |
| 涨 | 流出 | 拉高出货 |
| 跌 | 流入 | 逢低吸筹 |
| 跌 | 流出 | 加速下跌 |

#### OBV能量潮
- OBV创新高，股价未新高 → 底背离
- OBV创新低，股价未新低 → 顶背离
- OBV向上突破趋势线 → 买入
- OBV向下突破趋势线 → 卖出

---

### 四、技术指标

#### MACD
- DIF/DEA零轴上方: 多头市场
- DIF/DEA零轴下方: 空头市场
- 红柱变长: 上涨动能增强
- 绿柱变长: 下跌动能增强

#### KDJ
- J > 100: 超买区
- J < 0: 超卖区
- K、D > 80: 超买
- K、D < 20: 超卖

#### RSI
- RSI > 80: 超买
- RSI < 20: 超卖
- RSI(6)上穿RSI(12): 买入
- RSI(6)下穿RSI(12): 卖出

#### BOLL布林带
- 触及上轨: 超买，可能回调
- 触及下轨: 超卖，可能反弹
- 布林带收窄: 变盘在即
- 布林带开口: 趋势启动

## 数据处理

### MCP返回数据格式

MCP工具返回的K线数据为嵌套JSON `{"result": "{\"code\":0,...}"}`，需要二次解析：

```python
import json
raw = json.load(f)             # 第一层
data = json.loads(raw['result'])  # 第二层解析字符串
records = data['data']           # 提取数据数组
```

### K线数据字段映射

| API字段 | 含义 |
|--------|------|
| trade_date | 交易日期 (YYYYMMDD) |
| open_price | 开盘价 |
| high_price | 最高价 |
| low_price | 最低价 |
| close_price | 收盘价 |
| trade_amount | 成交量（股） |
| trade_balance | 成交额（元） |
| price_change_rate | 涨跌幅（小数） |
| ma5/ma10/ma20/ma60/ma120 | 均线 |
| dea | MACD DEA值 |
| macd | MACD柱状图值 |
| major_net_flow_in | 主力净流入（元） |
